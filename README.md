# RUNESCAPELIFEOS

RuneScape-inspired life operating system and habit tracker.

## Features

- XP and level system
- Daily quests
- Boss battles
- Streaks and habit decay
- Skill trees
- Heatmap tracking
- Focus timer
- Rewards shop
- Prestige system
- Hardcore mode
- Lore journal
- Seasonal progression
- RPG-style progression loop

## Tech Stack

- React
- Vite
- TailwindCSS
- LocalStorage persistence
- Netlify-ready deployment

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy to Netlify

Build command:

```bash
npm run build
```

Publish directory:

```bash
dist
```
